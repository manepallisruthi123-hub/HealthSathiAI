# HealthSathi AI

HealthSathi AI is a multilingual healthcare assistant built using Flask, SQLite, mBERT, Whisper, and gTTS.

## Model File

The file `model.safetensors` exceeds GitHub's file size limit and is therefore hosted separately via Google Drive.

Download the model file and place it inside:

healthsathi_mbert_model/

Then run:

python HealthSathi_Flask.py
